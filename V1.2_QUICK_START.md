#### 2. 贝叶斯推断 (简化版本)

```python
from circular_bias_detector.bayesian import simple_bayesian_bias

# 使用简化版本(不需要PyMC)
result = simple_bayesian_bias(
    psi_score=0.18,
    ccs_score=0.82,
    rho_pc_score=0.65
)

print(f"P(bias|data) = {result['bias_probability']:.3f}")

```

## 🐛 已知问题和解决方案

### 问题1: PyMC安装失败

**症状:** `pip install pymc` 报错

**解决方案:**
```bash
# 尝试PyMC3 (旧版本)
pip install pymc3==3.11.5 arviz theano-pymc

# 或使用conda
conda install -c conda-forge pymc
```

### 问题2: Transformers模型下载慢

**解决方案:**
```bash
# 使用国内镜像
export HF_ENDPOINT=https://hf-mirror.com

# 或手动下载模型
# 从 https://huggingface.co/models 下载
# 放到 ~/.cache/huggingface/
```

### 问题3: GPU不可用

**解决方案:**
```bash
# 检查CUDA
nvidia-smi

# 安装GPU版本的PyTorch
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```
